# Flex Developer Tools 💪

Three lightweight Python developer tools. Zero dependencies. Pure stdlib.

## 🕷️ Web Scraper
Extract data from any website. CSS selectors, link extraction, text extraction. CSV/JSON export.

```bash
python3 scraper.py https://news.ycombinator.com --selector ".titleline" --output titles.csv
python3 scraper.py https://example.com --mode links --output links.json
```

## 👁️ Site Monitor
Uptime checks, content change detection, multi-site monitoring.

```bash
python3 monitor.py check https://mysite.com
python3 monitor.py watch https://mysite.com --interval 30 --notify "Changed!"
```

## 🚀 API Builder
Turn any CSV or JSON file into a REST API in seconds. Filtering, search, sorting, pagination built in.

```bash
python3 apibuilder.py serve data.csv --port 8080
python3 apibuilder.py schema data.json
```
